# 🖥️ NOC Operations Dashboard — Network Performance Analytics

![Python](https://img.shields.io/badge/Python-3.10-blue) ![SQL](https://img.shields.io/badge/SQL-Window_Functions-lightgrey) ![Domain](https://img.shields.io/badge/Domain-Telecom_NOC-orange)

## Overview
Aggregates **alarm logs**, **site inventory**, and **field dispatch data** from three sources into a unified NOC analytics view. Computes MTTR by region, engineer response rankings, and alarm volume breakdowns.

## How to Run
```bash
pip install -r requirements.txt
python scripts/generate_data.py
python scripts/noc_analysis.py
```

## Key Outputs
- MTTR by region with SLA breach detection
- Engineer performance leaderboard
- Alarm type and severity breakdown
- Multi-source data integration pipeline
